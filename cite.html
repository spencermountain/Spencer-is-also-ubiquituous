this tool helps you make wikipedia citations.
If you're on a page that you'd like to cite in wikipedia, type 'cite', and it does its best to parse the information on the page into a proper reference tag.

it has a manual screen scraper for 40 or so popular news or reference websites.
it uses freebase.com to fill the 'publisher' field.
otherwise, it just grabs the url and the page's title, which is usually fine.

Usage: cite (if you select some text, it will quote it)
